## Description 
A user-friendly web application that empowers users to create professional CVs in a streamlined and efficient manner, within just two minutes.

## Features

<ul>
  <li><strong>Intuitive Interface:</strong> Provides a step-by-step guide and easy-to-use tools to craft compelling CVs.</li>
  <li><strong>Pre-built Templates:</strong> Offers a variety of professionally designed templates to cater to diverse needs and preferences.</li>
  <li><strong>Customization Options:</strong> Enables users to personalize their CVs with their unique information, skills, and experiences.</li>
  <li><strong>Authentication:</strong> Ensures secure access and management of user data through a robust authentication system.</li>
</ul>

<h1>Technologies Used</h1>
<ul>
  <li>DJANGO</li>
  <li>PostgreSQL</li>
  <li>JavaScript</li>
</ul>

## 🚀 Installation

1.Ensure you have Python (version 3.x recommended), Git, and Node.js installed on your system


#### 1. Clone the Repository:

```sh
git clone https://github.com/E.Roydev/2minuteCV.git
```

#### 2. Create a Virtual Environment (Recommended):

```sh
python3 -m venv venv
source venv/bin/activate # macOS/Linux
venv\Scripts\activate.bat # Windows
```

#### 3. Install Dependencies:
```sh
pip install -r requirements.txt
```

## ✔ Usage

#### In the project path write in Terminal

```sh
python manage.py runserver
```

This will typically run the server on http://127.0.0.1:8000/.
